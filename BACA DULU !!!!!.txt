Persiapan download Template web view praktikum :
1. install git ke device masing-masing
2. buat akun di https://github.com/
3. konfigurasi dengan cara : 
   $ git config --global user.name "John Doe"
   $ git config --global user.email johndoe@example.com
4. Download/clone template web view pada https://github.com/labrplitats/SI-Praktikum.git
5. Clone repository pada direktori : C:\xampp\htdocs

Push pada repositori masing-masing dengan cara:
1. Buat repository
2. Masuk pada folder program template web view
3. Buka git bash
4. Lakukan command seperti dibawah
   $ git remote remove origin
   $ git remote add origin nama_repository
   $ git push -u origin master

persiapan template web view :
1. Pastikan sudah terinstal Visual Studio Code & Instal XAMPP
2. Ekstrak lalu Taruh/Pindahkan Folder SI-Praktikum ke xampp > htdocs.
3. Agar Bisa dijalan kan, ubah semua extensi file (.html) menjadi (.php)
   Contoh : index.html > index.php
4. Start XAMPP (Apache & MySQL)
5. Buka Browser (Chrome Recommended) , lalu ketik di URL : http://localhost/SI-Praktikum/
6. Ubah Code File sesuai Comment di dalam setiap file. Contoh : <!-- Dirubah Saat Modul 2 -->.
7. LIHAT VIDEO YANG TERTERA AGAR LEBIH JELAS !!
